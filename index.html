D1 双轮足机器人自主上楼梯项目

本项目面向双轮足机器人在真实楼梯场景中的自主运动控制问题，围绕复杂接触切换、强非线性动力学以及关节安全约束开展控制算法设计与实机部署。 方法基于 NP3O 约束优化强化学习框架，在策略学习中显式引入关节级安全硬约束，并结合非对称 Actor-Critic、Barlow Twins 自监督表示学习与 Sim-to-Real 迁移流程，最终在本末科技 D1 双轮足机器人平台上实现了稳定自主上楼梯。

机器人平台：本末科技 D1
任务：自主上楼梯
方法：NP3O + 非对称 Actor-Critic
结果：真实环境稳定部署
项目简介 仿真视频 消融实验 实机实验

项目简介

楼梯场景中的双轮足机器人控制具有显著挑战性：机器人需要在离散台阶结构中完成连续推进，同时在接触切换过程中保持机体稳定，并满足关节级安全约束。 相比平地运动，该任务对策略的鲁棒性、动作连续性和实机部署能力提出了更高要求。针对传统控制方法在此类复杂场景中难以统一处理接触切换、动力学可行性与安全约束的问题，本项目构建了一套基于约束强化学习的控制方案，并完成了从仿真训练到真实机器人落地的完整流程。

项目目标

实现真实场景下的稳定自主上楼梯

在 D1 双轮足机器人平台上完成楼梯场景中的自主运动控制，实现机器人在真实环境中的连续、平稳且可重复的上楼梯表现。

算法框架

NP3O 约束强化学习控制

采用 NP3O 约束优化方法在策略训练中显式引入关节级安全硬约束，并结合非对称 Actor-Critic 结构与 Barlow Twins 自监督学习，提高策略的安全性、稳定性和泛化能力。

工程落地

完整 Sim-to-Real 部署流程

从仿真环境构建、策略训练到实机部署与调试，完成了完整的 Sim-to-Real 闭环，实现策略在真实楼梯环境中的稳定迁移。

方法概述：项目采用基于 NP3O 的约束强化学习框架，将关节安全限制作为训练中的硬约束进行优化；在策略结构上，使用非对称 Actor-Critic，让 critic 利用训练阶段的特权信息提升价值估计质量，而 actor 仅依赖部署时可获得观测进行决策；同时引入 Barlow Twins 自监督学习增强状态表征能力，提升策略在复杂楼梯场景下的鲁棒性与泛化性能；最终通过系统化的 Sim-to-Real 流程将训练策略稳定迁移到真实 D1 双轮足机器人上。

消融实验

本部分基于训练曲线与诊断指标分析关键模块对最终性能的影响。重点围绕 NP3O 约束优化、Barlow Twins 表征学习，以及约束项对训练稳定性和动作质量的调节作用展开。

关键模块对地形通过能力的影响

从 terrain level 曲线可以观察不同算法组合在训练过程中的地形适应能力。完整方法 NP3O+BT 在中后期保持了更稳定的增长趋势，最终达到较高的地形等级；相较于去除 Barlow Twins 的 NP3O 版本，加入表征学习后策略在复杂地形中的提升更平滑，说明更好的状态表征有助于提高样本利用效率与后期泛化能力。

Terrain Levels

地形等级训练曲线

曲线对比显示，HIMLoco 在较低难度地形附近较早饱和；PPO 系方法虽然部分阶段上升较快，但后期波动更明显。相比之下，NP3O+BT 在复杂地形上的上升过程更平稳，体现出约束优化和高质量表征对训练稳定性的共同作用。

Barlow Twins

Barlow Twins 表征分析

Barlow Twins 的 off-diagonal 项持续下降、effective rank 持续上升，说明特征冗余被逐步压缩，潜在表示维度被更充分利用；同时 latent std 的均值和最小值维持在合理区间，表明表征没有发生塌缩。这说明 BT 模块确实在训练中学到了更解耦、更稳定的状态编码。

Penalty Group A

接触与关节约束诊断

该组指标覆盖 joint acceleration、joint power 与 no-contact 等训练项。随着训练推进，各项约束惩罚逐渐进入更稳定区间，说明策略在提高通过能力的同时，也逐步学会抑制剧烈关节动作与无效接触行为。

动作平滑性与约束行为分析

除最终通过能力外，训练过程中的动作变化率、机体高度、速度跟踪误差和非期望接触等指标，也能更直接反映算法为何能够迁移到真实机器人上。整体来看，约束强化学习并不是单纯追求更快的地形升级，而是在稳定性、安全性和可部署性之间取得更好的平衡。

Penalty Group B

动作平滑性与机体姿态

action rate、角速度跟踪和 base height 等曲线反映了控制策略对动作平滑性与机体姿态的调节能力。完整方法在这些量上通常表现出更温和的收敛过程，说明其生成的控制信号更连续，更适合真实硬件执行。

Penalty Group C

速度跟踪与非期望接触

从线速度跟踪、undesired contacts 和 wheel velocity penalty 等指标可以看出，策略在学习推进能力的同时，也在逐步减少非期望碰撞和不稳定轮体行为。这是后续实现实机稳定上楼梯的重要前提。

Conclusion

算法分析结论

综合训练曲线可以得到一个比较清晰的结论：NP3O 提供了对安全约束与训练稳定性的基础保障，Barlow Twins 提升了状态表征质量和后期泛化能力，而各类约束项共同抑制了高频动作、异常接触和姿态失稳。正是这三个部分协同作用，才让策略不仅“能学会”，而且“能安全地迁移到真实 D1 机器人上”。

分析总结：如果只看单一任务进度指标，部分无约束或弱约束方法在早期可能上升更快；但结合表征质量、动作平滑性、接触稳定性和惩罚项收敛情况来看，完整方法在训练后期展现出更均衡的性能结构。这种“更稳而不是只更快”的优化特性，是双轮足机器人在真实楼梯场景中成功部署的关键。
