# d1-stairs-up

GitHub profile homepage for `d1-stairs-up`.

This repository hosts a static project page for the D1 stair-climbing robot project.

## Local preview

Open `index.html` in a browser.

## Deployment

Because this repository name matches the GitHub username, GitHub Pages will serve the homepage from:

`https://d1-stairs-up.github.io/`

## Notes

The current page references media assets under `images/` and `videos/`. Add those folders if you want the visual and video sections to render fully online.
